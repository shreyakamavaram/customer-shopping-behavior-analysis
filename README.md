# Customer Shopping Behavior Analysis

## Project Overview

This project analyzes customer shopping behavior using SQL and Power BI. The objective is to identify purchasing patterns, customer preferences, and revenue trends to generate meaningful business insights.

## Tools Used

- MySQL
- SQL
- Power BI

## Key Analysis

- Revenue Analysis by Gender
- Average Purchase Amount by Shipping Type
- Subscriber vs Non-Subscriber Spending
- Top-Rated Products
- Discount Impact Analysis
- Customer Segmentation
- Top 3 Products by Category
- Revenue by Age Group

## Key Insights

- Customer spending varies across different demographics.
- Discounts influence purchasing behavior and sales performance.
- Certain products consistently receive higher customer ratings.
- Subscribers show different purchasing patterns compared to non-subscribers.
- Product popularity varies across categories.

## Dashboard

The Power BI dashboard provides interactive visualizations for:
- Sales Performance
- Category Analysis
- Customer Segmentation
- Product Performance
- Revenue Trends
